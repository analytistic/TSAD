# RQTAD: 基于残差量化的时间序列异常检测

## 1. 概述

RQTAD 是一个基于**残差量化 K-Means (RQKMeans)** 的零样本时间序列异常检测模型。无需任何标签或梯度优化，在 TSB-AD-U 基准上达到 SOTA 性能。

本文档从**高斯过程 (GP)** 的视角，解释 RQKMeans 为什么有效、能检测什么、不能检测什么，以及检测能力的精确数学边界。

---

## 2. 高斯过程建模

### 2.1 信号模型

我们将时间序列建模为一个具有复合核函数的**高斯过程**的样本：

`x(t) ~ GP(0, K(t, t'))`

核函数由三部分组成：

`K(t,t') = K_smooth(t,t') + K_per(t,t') + K_noise(t,t')`

- **K_smooth** = Σ σ_i² · Matérn_νi(|t-t'|/ℓ_i)：Matérn 核，刻画平滑趋势
- **K_per** = Σ η_j² · exp(-2·sin²(π|t-t'|/p_j) / w_j²)：周期核，刻画重复模式
- **K_noise** = σ_ε² · δ_{t,t'}：噪声核，刻画观测噪声

| 核函数 | 作用 | 参数 |
|--------|------|------|
| Matérn K_ν | 平滑趋势、缓慢漂移 | 光滑度 ν、长度尺度 ℓ、方差 σ² |
| 周期核 K_per | 重复模式（日周期、周周期...） | 周期 p、宽度 w、方差 η² |
| 噪声核 | 观测噪声、微观波动 | 方差 σ_ε² |

光滑度为 ν 的 Matérn 核的谱密度为：

`S_Matérn(ω) ∝ (1 + (ℓω/2ν)²)^(-(ν+1/2))`

这意味着有限窗口协方差矩阵的**特征值多项式衰减**。

### 2.2 滑动窗口协方差

大小为 W 的滑动窗口在时刻 s 产生的随机向量：

`w_s = [x(s), x(s+1), ..., x(s+W-1)]ᵀ ∈ ℝ^W`

在 GP 先验下，`w_s ~ N(0, Σ)`，其中协方差矩阵具有 **Toeplitz 结构**：

`Σ_{ij} = K(i, j),  i, j = 0, 1, ..., W-1`

由于 K 只依赖于滞后 |i-j|，可写为 `Σ = Toeplitz(K(0), K(1), ..., K(W-1))`。

### 2.3 特征值分解与有效维度

设 `Σ = U Λ Uᵀ`，特征值为 `λ₁ ≥ λ₂ ≥ ... ≥ λ_W > 0`。

**定义（有效维度）：**

`d_eff = (Σ λ_i)² / (Σ λ_i²) = (tr Σ)² / ||Σ||_F²`

这是参与比的倒数：`d_eff ∈ [1, W]`，衡量"活跃"特征模态的数量。

对复合核，d_eff 近似分解为：`d_eff ≈ d_Matérn + d_periodic`

**Matérn 贡献：** 光滑度为 ν 的 Matérn 核，特征值衰减为 `λ_k ~ C·k^(-(2ν+1))`，超过阈值的特征值数量为：

`d_Matérn ~ (C / (ε·λ₁))^(1/(2ν+1))`

- 粗糙信号 (ν=1/2，指数核)：d_Matérn ~ O(W) — 接近满秩
- 光滑信号 (ν→∞，SE 核)：d_Matérn ~ O(1) — 非常低秩

**周期核贡献：** 每个周期为 p_j 的周期分量贡献 ⌊W/p_j⌋ 对主导特征值（对应 sin 和 cos 傅里叶模态）：

`d_periodic ≈ 2·Σ ⌊W/p_j⌋`

其余特征值处于噪声底 σ_ε²。

## 3. GP 流形上的 KMeans 量化

### 3.1 量化即有损压缩

KMeans 用 k 个码字 {c₁, ..., c_k} ⊂ ℝ^W 将 ℝ^W 划分为 **Voronoi 细胞**（每个细胞内的点都离该码字最近），每个窗口用最近的码字近似。均方量化误差为：

`Q(k) = E[min_j ||w - c_j||²]`

其中 `w ~ N(0, Σ)`。

### 3.2 基于特征值衰减的量化误差界

**定理 1（GP 量化界）：** 对 `w ~ N(0, Σ)`，最优 k-means 量化误差满足：

`Q(k) ≤ Σ_{i > d_eff(k)} λ_i`

**直观理解：** ℝ^W 中的 k 个码字最多张成 (k-1) 维仿射子空间。量化误差等于正交于该子空间的特征模态的能量。

**推论 1（Matérn 主导信号）：** 对特征值衰减 `λ_k ~ k^(-α)`（α = 2ν+1）的核：

`Q(k) ~ O(k^(-(α-1))) = O(k^(-2ν))`

| 光滑度 ν | 核类型 | 衰减率 α | Q(k) 速率 |
|----------|--------|----------|-----------|
| 1/2 | 指数核 | 2 | O(k⁻¹) |
| 3/2 | Matérn-3/2 | 4 | O(k⁻³) |
| 5/2 | Matérn-5/2 | 6 | O(k⁻⁵) |
| ∞ | SE (RBF) | ∞ | O(e^(-ck)) |

**关键洞察：** 信号越光滑，KMeans 收敛越快——需要的码字越少。

**推论 2（周期核主导信号）：** 对具有 d_per 个主导特征值（值为 η²）其余为 σ_ε² 的核：

- k < d_per 时：Q(k) ≈ (d_per - k)·η² + (W - d_per)·σ_ε²
- k ≥ d_per 时：Q(k) ≈ (W - k)·σ_ε²

一旦 k 超过周期特征模态的数量，误差在噪声底处饱和。

## 4. 残差量化：层次化分解

### 4.1 级联结构

RQKMeans 使用 L 个码本 C₀, C₁, ..., C_{L-1}，每个包含 k₀, k₁, ..., k_{L-1} 个码字。每级量化上一级的残差：

`ŵ = Σ c_l*`,  `c_l* = argmin_{c ∈ C_l} ||r_{l-1} - c||²`

其中 `r_{-1} = w`，`r_l = r_{l-1} - c_l*`。

### 4.2 为什么需要级联？GP 特征模态论证

**定理 2（层次化特征模态捕获）：** 在 GP 模型下，RQ 级联实现层次化特征模态分解：第 l 级捕获残差协方差 Cov(r_{l-1}) 的主导特征模态。L 级后的总量化误差满足：

`Q_cascade ≤ Σ_l Σ_{i > k_l} λ_i^(l)`

其中 `λ_i^(l)` 是 `Cov(r_{l-1})` 的特征值。

**证明概要：** 每级 KMeans 的码字张成 k_l 维子空间，捕获当前残差的前 k_l 个特征模态。剩余残差 r_l 的协方差特征值为尾部特征值 `{λ_i^(l)}_{i > k_l}`。由归纳法，总误差为各级尾部能量之和。 □

**复合核的分解：**

`Q_cascade ≤ [Σ_{i>k₀} λ_i^(0)] + [Σ_{i>k₁} λ_i^(1)] + [Σ_{i>k₂} λ_i^(2)] + ...`

其中 Level 0 捕获周期模态，Level 1 捕获平滑结构，Level 2 捕获噪声底。

**设计原则：** RQTAD 中递减的 k 列表 `[40, 20, 10]` 反映了这一层次：

| 级别 | 码字数 | 捕获内容 | GP 分量 |
|------|--------|---------|---------|
| 0 | k₀ = 40 | 主导周期特征模态 | K_per |
| 1 | k₁ = 20 | 次级平滑结构 | K_Matérn |
| 2 | k₂ = 10 | 残差精细结构 | 剩余 K_Matérn 尾部 + 噪声 |

### 4.3 乘积码本 vs. 贪心级联

**乘积码本** `C_prod = {c₀ + c₁ + ... + c_{L-1}}` 有 `K = Π k_l` 个码字，达到最优速率：

`Q_prod(K) ≤ Σ_{i > log₂ K} λ_i`

**贪心级联**（RQTAD 所用）是次优的，因为每级独立训练。但它达到：

`Q_cascade ≤ Σ_l Q_single(k_l)`

这是 **L 倍的多项式改进**，而非指数级。实际优势在于计算效率：训练 L 个大小为 k 的码本成本为 `O(L·k·N·W)`，而乘积码本为 `O(k^L·N·W)`。

## 5. 异常分数分析

### 5.1 分数定义

L 级量化后的异常分数为：

`A(w) = max_j | r_j^(L) - median(r^(L)) |`

其中 `r^(L) = w - ŵ` 是最终残差向量。

### 5.2 GP 零假设下的分数分布

**定理 3（正常分数分布）：** 对正常窗口 `w ~ N(0, Σ)`，L 级 RQ 后的残差近似为：

`r^(L) ~ N(0, Σ_res^(L))`

其中 `Σ_res^(L)` 的特征值由所有码本未捕获的尾部特征值组成。

异常分数 A(w) 是高斯随机向量在坐标轴上投影的 **range（极差）**。当残差分量近似独立同分布时：

`E[A(w)] ≈ σ_res · E[max_j |Z_j - median(Z)|]`

其中 `Z_j ~ N(0,1)` i.i.d.，`σ_res² = (1/W)·tr(Σ_res^(L))`。

对 W 个独立标准正态变量，期望最大绝对偏差为：

`E[max_j |Z_j - median(Z)|] ≈ √(2·ln W)`

因此：**`E[A(w)] ≈ σ_res · √(2·ln W)`**

### 5.3 检测阈值

实际中 RQTAD 使用所有窗口分数的 **MAD（中位绝对偏差）** 作为鲁棒阈值：

`threshold = median({A_i}) + γ · MAD({A_i})`

## 6. 能力边界

### 6.1 可检测性条件

**定理 4（最小可检测异常）：** 异常窗口 `w_anom = w_normal + δ` 可被 RQKMeans 检测到，当且仅当：

`||δ_⊥|| > τ_noise = √(Σ_{i > d_eff} λ_i) · (1 + γ)`

其中 δ_⊥ 是异常向量在码字所张成子空间的**正交分量**，τ_noise 由未解释特征模态决定的噪声底。

**含义：** 模型只能检测码本**未覆盖方向**上的异常。若异常完全在码字张成的子空间内，会被量化"吸收"，残差接近零。

### 6.2 不可检测的异常类型

| 异常类型 | \|\|δ_⊥\|\| | 可检测？ |
|---------|------------|---------|
| **相位偏移** w_anom = [x(s+Δ), ...] | ≈ 0 | **否** — 在同一流形上 |
| **幅度缩放** α·w | (α-1)\|\|w\|\| | 是，若 \|α-1\|·\|\|w\|\| > τ_noise |
| **高斯噪声** δ ~ N(0, σ_a²I) | σ_a·√W | 是，若 σ_a > τ_noise/√W |
| **点异常**（单点突变） | \|Δ\| | 是，若 \|Δ\| > τ_noise |
| **模式替换**（替换为另一正常模式） | ≈ 0 | **否** — 映射到另一个码字 |
| **频率偏移**（改变周期） | 取决于谱重叠 | 部分 — 若新频率不在码本中 |

### 6.3 相位与步长的关系

对具有周期 p 的主导周期核的 GP，步长为 δ 的相邻窗口的相关性为：

`corr(w_s, w_{s+δ}) = exp(-2·sin²(πδ/p) / w²)`

**步长 = 1 时：** 连续窗口高度相关，`corr ≈ 1 - 2π²/(w²p²)`（当 p >> 1）。

**有效独立样本数：** `N_eff ≈ N / gcd(δ, p)`

- δ 与 p 互素：N_eff = N/p — 覆盖所有相位，冗余最小
- δ = p：N_eff = N/p 但每周期只有一个相位 — 无相位多样性

**最优步长：** δ = 1 最大化信息（完整相位覆盖），冗余窗口无害，因为 KMeans 自然将它们归入同一聚类。

### 6.4 周期检测与 ACF 的联系

GP 的自相关函数中，周期分量在滞后 τ = p_j, 2p_j, 3p_j, ... 处产生**峰值**。RQTAD 中的 `_detect_period` 方法通过寻找这些峰值来设置窗口大小 `W = p_dominant`。

**为什么有效：** 设 W = p 确保 Toeplitz 协方差矩阵 Σ 恰好有 d_periodic 个主导特征值（对应周期核的傅里叶模态），KMeans 可以高效捕获。

### 能力边界（一句话）

> **RQKMeans 能检测任何在 GP 协方差的未解释特征模态上的投影超过量化噪声底 `σ_res·√(2ln W)` 的异常；它不能检测相位偏移或模式替换类异常，因为这些落在码字所张成的子空间内。**

## 8. RQKMeans 变体

### 8.1 PyramidRQKMeans

每级使用 SVD 降维。不在 ℝ^W 中聚类，而是将残差投影到前 d_l 个奇异向量子空间后再聚类。

**GP 解释：** 这显式地限制每级只捕获主导特征模态，充当**谱滤波器**。编码器 `V_{d_l}ᵀ` 投影到主导子空间，解码器 `V_{d_l}` 重建回原始空间。

### 8.2 PruneRQKMeans

训练时加入异常码字剪枝机制。每级训练后，通过异常相关比 `ρ_k = δ_k / ||c_k||²` 识别并掩蔽异常码字。

**GP 解释：** 这移除了捕获**分布外**模式（训练数据中的异常）的码字，防止码本"解释掉"异常。只有代表**主流 GP 行为**的码字被保留。

### 8.3 AlignMAD

量化前通过循环移位做相位对齐：`w_s → A(w_s)`。

**GP 解释：** 相位对齐将 Toeplitz 协方差转为**循环矩阵**。循环矩阵由 DFT 矩阵对角化，即特征模态恰好是傅里叶模态。这使周期核的贡献变为**秩-1**（一个特征值为 η²，其余为 σ_ε²），大幅降低有效维度，提高量化效率。

## 9. 非平稳性：为什么码本已经处理了它

### 9.1 核心洞察

非平稳性（均值偏移、方差变化、状态切换）**不需要**显式分段或局部码本拟合。KMeans 码本自然地将非平稳模式吸收进不同的 Voronoi 细胞：

- **状态 A**（低均值）→ 窗口聚集在码字 c_a 附近
- **状态 B**（高均值）→ 窗口聚集在码字 c_b 附近
- **状态 C**（高方差）→ 窗口分散在多个码字上

码本充当**状态字典**：它不关心模式*何时*出现，只关心*长什么样*。Voronoi 划分的时间不变性意味着非平稳性通过增加被占用的聚类数自动处理。

### 9.2 非平稳性真正伤害的地方：异常分数

码本能处理聚类，但**异常分数**有一个隐含假设：所有码字分配下的残差分布相同。

非平稳下，不同状态产生不同的残差分布：

`r | z*=k ~ N(0, Σ_k)`

其中 Σ_k 依赖于状态（码字）。当前分数 `A(w) = max_j |r_j - median(r)|` 忽略了这一点。高方差状态下的"正常"残差可能比低方差状态下的"异常"残差得分更高，导致：
- 高方差状态下的**误报**
- 低方差状态下的**漏报**

### 9.3 ~~候选方案：逐聚类分数归一化~~ [已废弃]

自然的修复是用聚类特定的期望分数归一化：

`A_norm(w) = max_j |r_j - median(r)| / E[max_j |r_j - median(r)| | z*=k]`

**结果：** 2026-04-22 实验失败。测试了两种变体：
1. 逐聚类 `σ_k = √(E[||r||²])`（L2 范数）→ 下降约 1 点
2. 逐聚类均值最大偏差（度量匹配）→ 下降约 1 点

**失败原因：** 分数度量（max-median）和归一化因子即使匹配，也可能不是正确的分解。逐聚类估计对小聚类也可能不稳定。此方向暂时废弃。

**建议下一步：** 从 **#5（分数重新设计）** 开始——换成在 GP 模型下有更好统计性质的度量，再重新审视归一化。或者从 **#1（加权 KMeans）** 开始——低风险且与分数设计独立。
